## 📖 Overview

This project implements a movie recommendation system using two collaborative filtering techniques: **User-Based Collaborative Filtering** and **Item-Based Collaborative Filtering**. The system is built in a Jupyter Notebook (`UserBased1.ipynb`) and uses the MovieLens dataset (`ratings.csv`) to predict movie ratings and generate personalized recommendations.

## 🧪 Methodology

The notebook is structured to compare two primary collaborative filtering approaches:

### 1️⃣ User-Based Collaborative Filtering
- **Objective:** Recommend items by finding similar users.
- **Process:**
    - A user-item matrix is constructed.
    - Cosine similarity is used to find users with similar rating patterns.
    - Predictions are generated by taking a weighted average of ratings from similar users.
- **Evaluation:** The model's performance is measured using Root Mean Squared Error (RMSE) and Mean Absolute Error (MAE).

### 2️⃣ Item-Based Collaborative Filtering
- **Objective:** Recommend items that are similar to items a user has liked.
- **Process:**
    - An item-user matrix is created.
    - Cosine similarity is calculated to find items with similar rating histories.
    - Recommendations are generated based on the similarity scores of items the user has rated.
- **Evaluation:** The model is evaluated using RMSE to measure prediction accuracy.

### 📊 Model Comparison
- The final section of the notebook directly compares the performance of the User-Based and Item-Based models.
- The RMSE scores are used to determine which approach is more effective for this dataset.
- Visualizations are provided to compare the recommendation scores from both techniques.
